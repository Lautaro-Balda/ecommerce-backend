# Ecommerce Backend

Backend de e-commerce construido con microservicios.

## Servicios
- **user-service** (Node.js): Registro, login, autenticación
- **product-service** (Go): Catálogo de productos
- **order-service** (Node.js): Órdenes y carrito
- **inventory-service** (Go): Stock y reservas
- **payment-service** (Go): Procesamiento de pagos
- **notification-service** (Node.js): Emails

## Cómo levantar el proyecto
\`\`\`bash
docker compose up
